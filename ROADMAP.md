
# Roadmap del Proyecto

## Versión 1 - MVP

### Objetivo

Contar con una aplicación funcional para:

- Consultar stock.
- Registrar ingresos.
- Registrar egresos.
- Controlar stock mínimo.
- Generar alertas.

### Módulos

#### Stock

- Visualizar inventario.
- Buscar insumos.
- Filtrar por categoría.

#### Ingresos

- Registrar entradas.
- Actualizar stock automáticamente.

#### Egresos

- Registrar retiros.
- Asociar empleado.
- Asociar sector.

#### Alertas

- Stock bajo.
- Stock agotado.
